Subscribe a user or agent to an issue (defaults to the caller)

USAGE
  multica issue subscriber add <issue-id> [flags]

FLAGS
  -h, --help            help for add
      --output string   Output format: table or json (default "json")
      --user string     Member or agent name to subscribe (defaults to the caller)

INHERITED FLAGS
  --help   Show help for command

LEARN MORE
  Use `multica <command> <subcommand> --help` for more information about a command.
