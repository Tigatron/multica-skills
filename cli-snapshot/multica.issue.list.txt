List issues in the workspace

USAGE
  multica issue list [flags]

FLAGS
      --assignee string   Filter by assignee name
  -h, --help              help for list
      --limit int         Maximum number of issues to return (default 50)
      --offset int        Number of issues to skip (for pagination)
      --output string     Output format: table or json (default "table")
      --priority string   Filter by priority
      --project string    Filter by project ID
      --status string     Filter by status

INHERITED FLAGS
  --help   Show help for command

LEARN MORE
  Use `multica <command> <subcommand> --help` for more information about a command.
