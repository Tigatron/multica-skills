Work with projects

USAGE
  multica project <command> [flags]

COMMANDS
  create:  Create a new project
  delete:  Delete a project
  get:     Get project details
  list:    List projects in the workspace
  status:  Change project status
  update:  Update a project

INHERITED FLAGS
  --help   Show help for command

LEARN MORE
  Use `multica project <command> --help` for more information about a command.
