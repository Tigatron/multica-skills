Work with agents

USAGE
  multica agent <command> [flags]

COMMANDS
  archive:  Archive an agent
  avatar:   Upload an avatar image for an agent
  create:   Create a new agent
  env:      Read and update an agent's custom environment variables (audited)
  get:      Get agent details
  list:     List agents in the workspace
  restore:  Restore an archived agent
  skills:   Manage agent skill assignments
  tasks:    List tasks for an agent
  update:   Update an agent

INHERITED FLAGS
  --help   Show help for command

LEARN MORE
  Use `multica agent <command> --help` for more information about a command.
