Read and update an agent's custom environment variables (audited)

USAGE
  multica agent env <command> [flags]

COMMANDS
  get:  Print an agent's custom_env as a JSON map (workspace owner/admin only; every call is recorded)
  set:  Replace an agent's custom_env (workspace owner/admin only; values equal to **** preserve the existing entry)

INHERITED FLAGS
  --help   Show help for command

LEARN MORE
  Use `multica agent env <command> --help` for more information about a command.
