Work with workspaces

USAGE
  multica workspace <command> [flags]

COMMANDS
  get:     Get workspace details
  list:    List all workspaces you belong to
  member:  Manage workspace members
  switch:  Set the default workspace for this profile
  update:  Update workspace metadata (admin/owner only)

INHERITED FLAGS
  --help   Show help for command

LEARN MORE
  Use `multica workspace <command> --help` for more information about a command.
